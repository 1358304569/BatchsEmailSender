=================项目简介=================
针对上海速优装饰有限公司每月的结算款、预算款邮件发送需求的工具，
用于批量发送邮件，每个邮件包括含收件人名字的附件

项目开始于：2018年8月25号
开发人员：小周
项目需求背景及特征可以参见本人CSDN博客：https://blog.csdn.net/qq_41059320/article/details/82754577
